# Desafio_APIs
Repositorio criado para resolução do Desafio de API utilizando Flask.

## Utilizando Flask
### Instalação

1. Crie um ambiente virtual:
    ```bash
    python3 -m venv .venv
    ```

2. Ative o ambiente virtual:
    ```bash
    .\.venv\Scripts\activate
    ```

3. Instale o Flask:
    ```bash
    pip install Flask
    ```

### Execução

Para rodar a aplicação, utilize o comando:
```bash
flask --app app run
